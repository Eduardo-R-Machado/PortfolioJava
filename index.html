Um aplicativo de sistema de faturamento desenvolvido em Java. Permite aos usuários gerar faturas, gerenciar clientes e acompanhar dados de vendas. Demonstra conectividade com banco de dados e operações CRUD usando JDBC.

Tecnologias utilizadas: Java, JDBC, MySQL.
